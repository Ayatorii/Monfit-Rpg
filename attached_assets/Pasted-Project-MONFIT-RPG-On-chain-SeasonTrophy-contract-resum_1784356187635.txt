- ERC-721 (OpenZeppelin via `forge install OpenZeppelin/openzeppelin-contracts`).
- Owner-gated: `mintTrophy(address player, uint256 season, uint8 rank) external onlyOwner returns (uint256 tokenId)`.
- Store `season`/`rank` per token (mapping `tokenId => TrophyData`).
- Simple on-chain-generated `tokenURI` (data URI or minimal JSON) — no need for hosted art assets for this hackathon.
