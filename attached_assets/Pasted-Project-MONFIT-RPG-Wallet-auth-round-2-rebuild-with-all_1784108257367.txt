3. **Cross-origin cookies**: Replit's preview pane runs the app inside an iframe on a different origin (replit.com), making all API calls cross-site. The session cookie MUST be set with `sameSite: "none", secure: true` (not `"lax"`), and EVERY fetch call to the backend (nonce, verify, session, logout) MUST include `credentials: "include"`. Missing this causes the nonce cookie to never reach the verify request, producing a false "invalid nonce" failure. Apply this from the first line of code, not as a later fix.
